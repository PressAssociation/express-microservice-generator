# Microservice Generator
> [Yeoman](http://yeoman.io) generator

## Getting Started

### Install Yeoman

```bash
npm install -g yo
```

### Install Microservice Generator

```bash
npm install -g generator-microservice
```
### Generate Project

Run the following command and follow the prompts

```bash
yo microservice
```

### Generate controller

```bash
cd project
yo microservice:controller beard
```
